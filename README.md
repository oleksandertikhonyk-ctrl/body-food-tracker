# Body Food Tracker

Повністю локальний PWA-трекер харчування та самопочуття українською. Дані зберігаються у `localStorage`, сервер і реєстрація не потрібні.

## Запуск

```bash
npm install
npm run dev
```

Production-збірка:

```bash
npm run build
npm run preview
```

Після відкриття у браузері застосунок можна встановити на домашній екран. Service worker та manifest генеруються автоматично під час збірки.

## Можливості

- onboarding і автоматичні цілі за Mifflin–St Jeor;
- 100+ продуктів, власні продукти, пошук, обране та архів;
- додавання їжі за порцією, приблизною вагою або природним текстом;
- вода, вага, обхват живота, здуття та активність;
- щоденник, історія й графіки з реальних локальних даних;
- експорт/імпорт JSON та повне очищення;
- офлайн PWA, mobile-first інтерфейс.
